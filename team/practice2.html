보완점 설명
header 부분이 없음 웹사이트의 주요 내용을 소개하는 부분이 없음
nav를 통해 "지국찾기", "선생님 지원", "자료 다운로드" 등 링크들이 태그로 그룹화해야할듯 웹사이트의 주요 내비게이션 링크들이 그룹화되어 있지 않음
div를 주로 사용하여 main 부분이 명확하지 않음 웹사이트의 주요 내용을 담는 부분이 명확하게 구분되어 있지 않음
section을 통하여, "웅진 씽크빅 라운지" 같은 팝업 모달들을 정의하면 좋을것같음 웹사이트의 특정 섹션들이 명확하게 구분되어 있지 않음
article을 사용할 부분이 보이지않음 독립적으로 구분될 수 있는 콘텐츠가 명확하게 구분되어 있지 않음
"Family Site" 같은 경우는 메인과는 동떨어진 컨텐츠로 보이기때문에 aside가 적합해보임 웹사이트의 메인 콘텐츠와는 별개의 부가적인 정보가 명확하게 구분되어 있지 않음
footer 부분을 굳이굳이 div로 해둔것같음 웹사이트의 하단 정보가 명확하게 구분되어 있지 않음
meta태그의 부재 meta description의 부재로 인하여 검색엔진 노출면에서 아쉬움
button태그를 사용해서 페이지의 이동을 처리 a태그를 이용해서 페이지 내에서도 이동을 처리하면 더 의미에 맞게 됨
이유 설명
