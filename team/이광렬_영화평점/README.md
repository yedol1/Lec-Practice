# <span id='top'>🎬 영화 검색 사이트</span>

## 1. 서비스 소개

**영화 검색 서비스입니다.**
YTX.MX 의 Open API 를 통해, 영화검색 사이트를 구현해 보았습니다.

해당 프로젝트를 실행하고 싶다면,
다운받으시고

```
npm i -> npm start
```

를 해주시면 됩니다.

## 2. 프로젝트 구조

```
📦 src
   ├─ 📂 api
   │  ├─ 📜 index.js
   │  └─ 📜 movie.js
   ├─ 📂 components
   │  └─ 📜 Card.js
   ├─ 📜 App.js
   └─ 📜 index.js
```
